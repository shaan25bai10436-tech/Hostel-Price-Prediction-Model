# Basic Hostel-Price-Prediction-Model

Buying or selling a house is difficult because prices depend on many factors like size, location, and number of rooms. It is not easy for a person to estimate the correct price.  This project aims to build a simple machine learning model that can predict house prices based on basic features.

This project aims to build a simple machine learning model that can predict house prices based on basic features. The goal is to understand how data can be used to make better decisions.

# Why It Matters

Housing is one of the most important financial decisions in a person’s life. A prediction model can help buyers and sellers get a rough estimate of property prices. It also demonstrates how machine learning can be applied to real-world problems.

# Approach

1.Loaded the dataset and explored important features
2.Selected relevant numerical features such as rooms, distance, and building area
3.Handled missing values by removing incomplete rows
4.Split the dataset into training and testing sets
5.Trained models like Decision Tree and Random Forest
6.Compared performance using Mean Absolute Error (MAE)

# Results

The Random Forest model performed better than the Decision Tree model and gave more accurate predictions.

The model was able to capture relationships between features such as building area and distance from the city.

This shows that machine learning can be useful in predicting house prices with reasonable accuracy.

# What I Learned

Data cleaning is an important step before training models
Choosing the right features improves prediction accuracy
Random Forest performs better than simple Decision Trees
Machine learning can solve real-world problems like price prediction
Small changes in data handling can affect results significantly
